    HashCatch, a Document Similarity Engine

    Plagiarism detection built entirely from scratch in C++, without STL. Implements a polynomial rolling hash and a custom open-addressing hash table with linear probing for O(1) word lookup. Applies Jaccard similarity across two hash tables to produce a document similarity percentage. CLI tool accepting any two text files.
